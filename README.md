# Harmony Basket Center

Bienvenue dans le dépôt du projet Harmony Basket Center.

## Branches

- **main** : Branche de production (stable)
- **dev** : Branche de développement actif

## Description

Ce dépôt contient le code source pour le projet Harmony Basket Center.

## Démarrage

Pour cloner le dépôt :
```bash
git clone https://github.com/Ressler18/Harmony-Basket-Center.git
cd Harmony-Basket-Center
```

Pour basculer vers la branche de développement :
```bash
git checkout dev
```
